### Monte Carlo simulation
Runs on the veracity of Law of Large Numbers. If increasing the number of samples produces more and more accurate estimation of a measure, just run the entire scenario over and over again, with all possible values accounted for over the runs. E.g. if FD interest rates are always between 9% and 3%, and you want the most likely rate over a period of 10 years, just run the simulation for all possible interest rate values for 10 years, then compare the graphs. \
Monte Carlo simulation is valuable because you can do this analysis for any number of criteria / multiple dimensions, and the accuracy depends only on number of observations, n. $accuracy \propto \frac{1}{\sqrt{n}}$.


### t-test
Using sample average to estimate population average. Used when sample size are low, and low accuracy works.

### z-test
Using multiple sample averages to estiamte population average. Used when accuracy needs to be high, but needs large sample set.
